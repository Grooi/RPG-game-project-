#程式如何安裝執行(圖文講解):
請先去https://www.sfml-dev.org/download.php 下載sfml
並解壓後放在D槽，若是沒有D槽則改放C槽，找到環境配置中的V/C++與C/C++當中有SFML的項目並將其中的D:改成C:
